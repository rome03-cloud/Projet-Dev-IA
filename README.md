# 🚀 Portfolio Interactif - Pacome EKRA

Bienvenue sur mon projet de présentation personnelle. Ce projet est une "Landing Page" (page d'atterrissage) développée pour mettre en avant mon profil de développeur Frontend, mes passions et mes objectifs professionnels.

## 📝 Description
Ce projet consiste en une interface moderne et responsive comprenant :
- Un **Hero Header** avec des effets visuels de lueurs (orbes flous).
- Une typographie soignée utilisant la police **Roboto**.
- Une structure optimisée pour le web avec **Tailwind CSS**.
- Une interactivité fluide gérée en **JavaScript**.

## 🛠️ Technologies utilisées
- **HTML5** : Pour la structure sémantique de la page.
- **CSS3 & Tailwind CSS** : Pour le stylisme moderne, les animations de flottement et le responsive design.
- **JavaScript (Vanilla)** : Pour la gestion des événements et l'interactivité du bouton.
- **Google Fonts** : Intégration de la police Roboto.

## 📁 Structure des fichiers
- `index.html` : Contient l'ensemble du code (Structure, Style et Logique).
- `README.md` : Documentation du projet (ce fichier).
